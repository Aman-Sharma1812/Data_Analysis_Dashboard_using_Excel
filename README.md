# Data_Analysis_Dashboard_using_Excel
## Project Object
The objective is to analyze sales and profit performance across different states, product categories, sub-categories, and time periods.
The dashboard helps identify profitable segments, loss-making products, sales trends, and key business insights to support data-driven decision-making.

## Project Description
This project analyzes sales and profit data to evaluate business performance across products, regions, and time. An interactive dashboard highlights key KPIs, trends, and profit patterns. The analysis supports data-driven decision-making by identifying profitable and loss-making areas.
